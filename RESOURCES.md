# Email OTP Testing Resources

## Knowledge

- [Resend: 403 Error Using resend.dev Domain](https://resend.com/docs/knowledge-base/403-error-resend-dev-domain)
  อธิบายว่า `resend.dev` ใช้ทดลองและส่งได้เฉพาะอีเมลที่ผูกกับบัญชี Resend
- [Resend: Create API Key](https://resend.com/docs/api-reference/api-keys/create-api-key)
  ใช้ตรวจสอบประเภทสิทธิ์ของ API Key; งานนี้เลือก `sending_access`
- [Supabase: Environment Variables](https://supabase.com/docs/guides/functions/secrets)
  วิธีเก็บ Secret ของ Edge Functions ทั้งบนเครื่องและ Production

## Wisdom (Communities)

  ใช้ค้นปัญหา Edge Functions และ Secret ที่พบจากผู้ใช้งานจริง

  ใช้เมื่อบัญชีถูกจำกัดการส่งหรืออีเมลทดลองไม่เข้าตามสถานะใน Dashboard
